# emoji-proofer
A service to prove what you really meant.

## Usage (with Docker):
```bash
git clone https://github.com/rubbyhorn/emoji-proofer.git
cd emoji-proofer
docker build -t emoji-proofer . && docker run -it --rm -p 1234:1234 emoji-proofer
```
